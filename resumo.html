Resumo de jQuery

Base:

É uma biblioteca e precisa ser importada. Ter na mesma pasta do projeto os aquivos jquery-3.7.0.min.js, jquery-ui.js e a pasta de arquivos jquery-ui-1.13.2 que são facilmente encontrados para download. No documento HTML, adicionar o seguinte código com estrutura base na tag head:

            
                <!-- jQuery -->
                    <script src="jquery-3.7.0.min.js"></script>
                    <script src="jquery-ui.js"></script>
                    <!-- Script -->
                    <script>
                    // Ao carregar o documento
                    $(document).ready(function(){
                        // Código aqui
                    });
                    </script>

Seletores:

É possível criar variáveis dentro da estrutura base com partes selecionadas do documento.

            
                    var campo = $("input").val();
                    var titulo = $("#titulo").html();
                    var sub = $(".subtitulo").html();

Onde .val é equivalente de value, e .html é equivalente a innerHtml. input se refere a tag input no HTML, #titulo se refere ao id e .subtitulo a classe em HTML.

É possível utilizar os campos selecionados chamando as variáveis ou apenas seguinto a sintaxe mostrada nos códigos dentro desse documento.

            
                alert(campo);
                alert(titulo);
                alert(sub);

Eventos:

Os eventos no jQuery não são dentro do HTML, o que deixa o códiga um pouco mais "limpo", considerando que esses códigos estão dentro da tag head.

            
                $(document).ready(function(){

                    // Evento de clique
                    $("button").click(function(){
        
                        // Adicionar um texto no H1
                        $("h1").html("Você clicou!");
        
                    });
                });

Onde click é equivalente a onclick.

Efeitos:

Exibir e Ocultar:

Foram criados dois buttons com id exibir e ocultar. A img some e aparece ao clicar nos botões.

            
                $(document).ready(function(){

                    // Exibir
                    $("#exibir").click(function(){
                        $("img").show();
                    });
        
                    // Ocultar
                    $("#ocultar").click(function(){
                        $("img").hide();
                    });
                });


Fade:

            
                $(document).ready(function(){
                    $("h1").css("display", "none");
        
                    $("#exibir").click(function(){
                        $("h1").fadeIn("slow");
                    });
        
                    $("#ocultar").click(function(){
                        $("h1").fadeOut(3000);
                    });
                });

Onde o tempo default é 0,4s, o slow é 0,6s e o 3000ms = 3s.



Aprendendo JQuery

Slide:

O efito slide pode ser usado com o indicativo de tempo ou sem.

            
                $(document).ready(function(){
                    $(".ocultar").click(function(){
                        $(".mensagem").slideUp();
                    });
        
                    $(".exibir").click(function(){
                        $(".mensagem").slideDown(200);
                    })
                });

Notar que para exibir é slideDown e ocultar é o Up.

Animate:

Aqui se pode alterar da forma que imaginar, basta usar a criatividade.

            
                $(document).ready(function(){
                    $("button").click(function(){
        
                        $("div").animate({
                            width: "400px",
                            backgroundColor: "#ff0000"
                        },1000);
                    });
                });

Essa parte é a que precisa da chamada do arquivo jquery-ui.js.

Manipular CSS:

            
                $(document).ready(function(){
                    $("h1").css({"color": "red",
                        "borderStyle": "solid",
                        "borderColor": "green"
                    });
                });

Alterar atributos:

Aqui, as duas imagens foram colocadas no arquivo exatamente iguais, mas uma foi alterada utilizando jQuery.

            
                $(document).ready(function(){
                    $("img").attr("width","400");
                    $("img").attr("title","gay chorando");
